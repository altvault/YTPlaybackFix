# YTPlaybackFix
<p><strong>Fix playback issues by refreshing the video every time the error 14 appears</strong> (the error that for some accounts appear every 50/60 second playing a video on a YouTube sideloaded app on iOS)</p>
<p><strong>Latest release: <a href="https://github.com/Mark02-2012/YTPlaybackFix/releases/tag/v0.2">0.2</a></strong>, please test and report any issue <a href="https://github.com/Mark02-2012/YTPlaybackFix/issues">here</a></p>
<p><strong>IMPORTANT please if you want to integrate this tweak into your mod make sure you put the step to clone the branch "main" to build the stable version and "beta-testing" to build the latest beta (at your own risk)</strong></p>
